# Persona cá nhân và công ty

## Purpose

This folder captures the working persona, company context, and transformation context used to guide the Super AI Transformation Platform / BD Harness initiative.

The content is intentionally written as a **sanitized working context** for planning, consulting discovery, requirement definition, and MVP execution. It should not be treated as confidential company documentation.

## Confirmed context

```text
Company context: FPT Japan / FPT Software Japan context
User role: Middle manager
Team scope: Department / unit of approximately 80-100 people
Transformation focus: Build an AI-assisted delivery solution for the department
Primary beachhead: Requirement -> RCD -> Basic Design transformation
```

## Core positioning

```text
This is not a chatbot initiative.
This is a governed AI-assisted delivery transformation initiative for a Japan enterprise software delivery context.
```

```text
A middle manager in FPT Japan can create high leverage by building a governed AI-assisted delivery capability for an 80-100 person department, starting with RD -> RCD -> Basic Design, then scaling into reusable workflows, skill library, validation, traceability, and human-gated artifact generation.
```
