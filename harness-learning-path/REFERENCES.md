# References — Harness Learning Path

This file tracks the concrete learning sources used for Week 01.

---

# 1. Anthropic Academy

URL: https://www.anthropic.com/learn

Use for:

- AI Fluency
- Claude foundation
- Claude Code
- MCP
- Agent skills
- Certificates on completion where available

Why relevant:

Anthropic Academy is useful for learning the design patterns behind Claude-based agent workflows, especially context, tools, MCP, Claude Code, and agent skills.

---

# 2. Claude Code in Action

URL: https://anthropic.skilljar.com/claude-code-in-action

Use for:

- Coding assistant architecture
- Workspace-based agent execution
- Context management
- Custom commands
- MCP servers
- GitHub integration
- Hooks and SDK
- Quiz / completion certificate where available

Why relevant:

This is the most relevant course for understanding artifact-centric architecture, where files, Git diffs, tests, and repo structure act as the working state of the agent.

---

# 3. Microsoft AI Agents for Beginners

URL: https://github.com/microsoft/ai-agents-for-beginners

Use for:

- Agent design basics
- Multi-agent concepts
- Tool use
- Agentic RAG
- Semantic Kernel / AutoGen ecosystem orientation

Why relevant:

Good for understanding Microsoft-style agent patterns and enterprise AI agent terminology.

---

# 4. Microsoft AI Agents: From Foundations to Applications Professional Certificate

URL: https://www.coursera.org/professional-certificates/microsoft-ai-agents

Use for:

- Microsoft Azure AI agent development
- Multi-agent systems
- Enterprise deployment
- Security and responsible AI
- Portfolio projects
- Shareable certificate

Why relevant:

This is the strongest certificate-oriented option for enterprise agent development.

---

# 5. DeepLearning.AI — AI Agents in LangGraph

URL: https://www.deeplearning.ai/courses/ai-agents-in-langgraph

Use for:

- LangGraph components
- Agent from scratch
- State management
- Persistence
- Human-in-the-loop
- Agentic search

Why relevant:

Useful for understanding state-centric orchestration and how LangGraph structures controllable agents.

---

# 6. LangChain Academy — Introduction to LangGraph

URL: https://academy.langchain.com/courses/intro-to-langgraph

Use for:

- LangGraph foundation
- StateGraph

Official LangChain/LangGraph training source for stateful workflow orchestration.
